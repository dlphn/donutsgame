# donutsgame
A simple multiplayer text-based adventure game.

## Files to open:
* 0_installDependencies OR  
```shell
      $ npm install
```
* 1_startMongo (verify MongoDB version)
* 2_prefill_DB_with_rooms
* 3_runServer OR
```shell
      $ npm start
```
* 4_localhost

One time play. When you quit, you lose your bag and your counter.
